# EASG-PIA-InvOP

En el repositorio se encuentra el código usado para la resolución del caso de estudio "Ruta Mínima para Despliegue de Cableado Subterráneo de Fibra Óptica en un Parque Industrial.

A través de la librería Networkx, se crea un grafo con los datos adyacentes al parque industrialm, tras lo cual se calcula con el algoritmo de Kruskal el arbol de mínima expansión que logre un despliegue del cableado de fibra óptica óptima donde se minimice el presupuesto usado

Librerias requeridas para el funcionamiento del programa:
- Networkx
- MatplotLib
